# Secure Connected Car System

## Overview

The Secure Connected Car System is a personal project focused on developing a containerized platform for collecting, processing, and displaying vehicle telemetry data.

The project explores the use of Docker-based services, Raspberry Pi hardware, and OBD-II vehicle interfaces to create a modular system capable of monitoring real-time automotive metrics through a local dashboard.

Security is a core design objective. Planned features include authentication, audit logging, secure communications, and container isolation.

## Technologies

- Docker Desktop
- Docker Compose
- Git & GitHub
- Raspberry Pi 4 (planned)
- OBD-II Diagnostics (planned)
- Python (planned)
- Nextcloud (planned)

## Project Status

Current development is focused on system architecture, documentation, containerized deployment planning, and security design.

## Planned Features

- Vehicle telemetry collection
- Local dashboard visualization
- OBD-II integration
- Raspberry Pi deployment
- Authentication system
- Audit logging
- Secure communications
